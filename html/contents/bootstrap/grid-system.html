    BOOSTRAP GRID SYSTEM

    Bootstrap Grid System is a concept that was created to provide quick, easy and efficient way to develop a responsive website layouts. It also predefined numerous grid classes for making grid layouts for different devices such as cellphones, tablets, desktops and laptops, etc.

    Bootstrap’s Grid System is made up of 3 things:

    1. Container
    2. Row
    3. Column

    Container

    Every Bootstrap grid system must have a container in order to create rows and columns. A container consists of a simple <div> element with a class of .container. The container is used to provide a proper width for the layout, acting as a wrapper for the content. The image below is an example of creating a container:

    Row

    Row acts like a wrapper around the columns. The purpose of creating a row is to keep the layouts’ columns inside from overflowing onto the other rows.

    The rows extent from left side to right side of the container element. It is created by adding the class .row to a block level element inside a container.
