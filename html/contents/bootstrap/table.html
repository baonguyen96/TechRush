    BOOTSTRAP TABLE

    Tables are presented in almost every application platforms (web, desktop or mobile app) and they are a key element in delivering data to the end user. The Bootstrap tables are actually used to present information in framework system consists of columns and rows. With the up-to-date Bootstrap 4 framework, you are able to easily improve the appearance of the table.

    Nowadays, considering how important the user interface is, understanding the way to enhance the look of a Bootstrap tables class comes to be quite relevant. In this case, one of the systems that have become reference is Bootstrap. This front-end framework, within many other features , provides a number of functions for styling and optimizing the display of different components, such as tables.

    Example of creating a Bootstrap table with borders

    Condense Table

    In a situation where you want to make your table tidy and taking up a little less space, you can condense your table or cut cell padding in half through this class: .table-condensed.

    Keep this in mind, while Bootstrap 3 uses .table-condensed to condense a table, Bootstrap 4 uses .table-sm. Both of these cut cell padding in half.

    Example of creating a Bootstrap condensed table
