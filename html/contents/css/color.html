    Colors

    Naming Conventions (HTML5 Compatible Color Names): provide an orderly structure for the color names.

    Hexadicimal Code

    A hexadecimal color is specified with: #RRGGBB. RR (red), GG (green) and BB (blue) are hexadecimal integers between 00 and FF specifying the intensity of the color. For example: #0000FF displayed as blue.

    RGB

    An RGB color value is specified with: rgb(red, green, blue). Each parameter (red, green, and blue) defines the intensity of the color as an integer between 0 and 255. For example: rgb( 0, 0, 255) displayed as blue.

    RGBA

    RGBA color values are an extension of RGB color values with an alpha channel - which specifies the opacity for a color. An RGBA color value is specified with: rgba(red, green, blue, alpha). The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (fully opaque). For example: rgba(255, 0, 0, 0.3) displayed red with opacity

    HSL

    HSL color values are supported in IE9+, Firefox, Chrome, Safari, and in Opera 10+. HSL stands for hue, saturation, and lightness. HSL color values are specified with: hsl(hue, saturation, lightness)

    HSLA

    HSLA color values are an extension of HSL color values with an alpha channel - which specifies the opacity of the object. An HSLA color value is specified with the hsla() function, which has the following syntax: hsla(hue, saturation, lightness, alpha)
