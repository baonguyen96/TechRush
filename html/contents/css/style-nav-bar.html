    Style Navigation Bar

    Navigation Bars:

    a list of links, using ul and li

    Vertical Navigation Bar: style the <a> elements inside the list.
    For example:
    li a {
    display: block;
    width: 60px;
    }

    Horizontal Navigation Bar: using inline or floating list items
    For example:
    li {
    display: inline;
    }
    li {
    float: left;
    }
