    Images

    Filling your website with images can help with adding personality and design to all of your content. Images can be used in a variety of ways by mixing it with other HTML syntax, however, we are only going to be focusing on adding a simple image to your web page.

    Image HTML Tags

    The proper format for an image tag is <img src="#"> with the "#" being the img source file.

    The alt attribute is important when defining an image because it provides alternate text for the image on the web page if the for some reason it cannot be viewed. 

    Example:
<img src="#" alt="Broken Image">

    Outputs:

    Tag Used
    Definition
    <img src="" alt="">
    Used for defining an image
