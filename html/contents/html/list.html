    Lists

    Using HTML, we can create various types of lists in order to more accurately organize and format information. In this section we'll be looking at both Unordered and Ordered lists. Each type of list provides a different structure output with the intent of accomplishing different purposes.

    Unordered Lists

    Using the <ul> tag we can begin an Unordered List. This initiates a list that utilizes bullet points for each list item. List items are specified using the <li> tag. 

    Example:
<ul>
    <li> List Item 1 </li>
    <li> List Item 2 </li>
    <li> List Item 3 </li>
<ul>

    Outputs:

    • List Item 1
    • List Item 2
    • List Item 3
    Ordered Lists

    Using the <ol> tag we can begin an Ordered List. This initiates a list that utilizes numbered bullet points for each list item as opposed to the un-numbered bullet points. Using the same <li> tag, list items can be specified for the overall list. 

    Example:
<ol>
    <li> List Item 1 </li>
    <li> List Item 2 </li>
    <li> List Item 3 </li>
</ol>

    Outputs:

    1. List Item 1
    2. List Item 2
    3. List Item 3



    Tag Used
    Definition
    <ul>
    Used for establishing an Unordered List
    <ol>
    Used for establishing an Ordered List
    <li>
    Used to define the different list items
