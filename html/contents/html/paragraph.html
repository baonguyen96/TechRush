    Headings and Paragraphs

    In this section, we'll be covering how to add the main text to a webpage you're working on. Some of the basic HTML elements used in a document involve "headings" and "paragraphs".

    Headings

    Headings are used to properly identify each section of text or media located on a web page. Currently 6 different headings can be used on a web page with varying text sizes depending on the type of heading applied to it. Starting with the biggest and most important heading, <h1>, and going to the smallest and least important, <h6>, with the various sizes in between (<h2>, <h3>, <h4>, & <h5>)
    It should be noted that while it may be possible to use different heading styles for text decoration, avoidance of this habit is important to ensure proper HTML syntax

    Here are some examples of headings

    This is an "h1" heading

    This is an "h2" heading

    This is an "h3" heading

    This is an "h4" heading

    This is an "h5" heading
    This is an "h6" heading

    Headings are great to use for web page organization as well as content navigation. The structure of your content is essentially set up by your headings and helps search engines identify the different parts of your content

    Paragraphs

    Using the <p> tag we can define different paragraphs in the content you have set up on your web page.
    One thing to note about the content you put inside of the paragraph tags, is that it will not be affected by carriage returns or tab inputs. To add carriage returns to text-based content one can utilize the <br> tag.

    Another work around is by using the <pre> tags. Using a <pre> tag will help with sustaining any space and/or line breaks in the html code

    For Example this:

                <pre>
                This is line 1

                This is line 2

                This is line 3
            <pre>

    Will Output this:

                    This is line 1

                This is line 2

                This is line 3


    Tag Used
    Definition
    <h1>
    Used for defining an h1 heading
    <h6>
    Used for defining an h6 heading
    <p>
    Used to define a paragraph
    <br>
    Used for initiating a line break
    <pre>
    Helpful with keeping spaces and line breaks intentionally added to text
