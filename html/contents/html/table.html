    Tables

    Tables are helpful when a web page calls for proper formation of data or information into columns and rows. The main HTML tag for table initiation is the <table> tag. There are three HTML tags needed inside the table tags when setting up a table on your web page; <th>, <tr> and <td>.

    Table Headers

    Table headers are defined with the <th> tag and establishes the header for each column of the table.

    Table Rows

    The <tr> tag is used in order to create a new row in the table.

    Table Data

    The table data tag, <td>, is used to iniate the data inside each cell of the table. The data can be text, images, links and so on.

    Putting it all together will look like this: 

    <table>
<th>Header 1</th>
<th>Header 1</th>
<tr>
    <td>Row 1, Col 1</td>
    <td>Row 1, Col 2</td>
</tr>
<tr>
    <td>Row 2, Col 1</td>
    <td>Row 2, Col 2</td>
</tr>
</table>

    Full Table Output:

    Header 1 Header 1
    Row 1, Col 1 Row 1, Col 2
    Row 2, Col 1 Row 2, Col 2

    Tag Used
    Definition
    <table>
    Used for initiating a table
    <th>
    Used for defining a table header
    <tr>
    Used for defining a table row
    <td>
    Used for defining table data
